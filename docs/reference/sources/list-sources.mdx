---
title: "List sources"
description: "List all of your data sources"
api: "GET http://localhost:3000/sources"
auth: "bearer"
---

### Response

<ResponseField name="content" type="array of objects">
<Expandable title="Toggle object">

<ResponseField name="nickname" type="string"></ResponseField>

<ResponseField name="schema" type="string"></ResponseField>

<ResponseField name="database" type="string"></ResponseField>

<ResponseField name="status" type="string">
  Either `REACHABLE` or `UNREACHABLE`
</ResponseField>

<ResponseField name="sourceType" type="string">

`MYSQL`, `POSTGRES`, `SNOWFLAKE`, `REDSHIFT`, `BIGQUERY`

</ResponseField>

</Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --request GET \
     --url http://localhost:3000/sources \
     --header 'Accept: application/json'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "content": [
    {
      "nickname": string,
      "schema": string,
      "database": string,
      "status": "REACHABLE" | "UNREACHABLE",
      "sourceType": "MYSQL" | "POSTGRES" | "SNOWFLAKE" | "REDSHIFT" | "BIGQUERY",
    }
  ]
}
```

</ResponseExample>
